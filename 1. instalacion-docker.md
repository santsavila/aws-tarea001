## INSTALACIÓN Y VERIFICACIÓN DE DOCKER EN UBUNTU EC2 AWS

1. Primero se actualizó el sistema operativo Ubuntu de la instancia EC2:

sudo apt update
sudo apt upgrade -y

2. Luego se instaló Docker con el siguiente comando:

sudo apt install docker.io -y

3. Una vez instalado Docker, se inició el servicio:

sudo systemctl start docker

4. Después se habilitó Docker para que se inicie automáticamente cada vez que la instancia EC2 se reinicie:

sudo systemctl enable docker

5. Para verificar que Docker se instaló correctamente, se ejecutó:

sudo docker --version

El resultado obtenido fue:

Docker version 29.1.3, build 29.1.3-0ubuntu4.1

Esto confirma que Docker quedó instalado en la instancia Ubuntu.

6. Para comprobar que Docker funciona correctamente, se ejecutó un contenedor de prueba:

sudo docker run hello-world

El resultado mostró el mensaje:

Hello from Docker!
This message shows that your installation appears to be working correctly.

Esto confirma que Docker pudo descargar una imagen desde Docker Hub y ejecutar correctamente un contenedor.

7. Después se instaló Docker Compose con el comando:

sudo apt install docker-compose -y


8. Para verificar Docker Compose V2, se ejecutó:

docker compose version


Comandos principales utilizados:

sudo apt update
sudo apt upgrade -y
sudo apt install docker.io -y
sudo systemctl start docker
sudo systemctl enable docker
sudo docker --version
sudo docker run hello-world
sudo apt install docker-compose -y
docker compose version
